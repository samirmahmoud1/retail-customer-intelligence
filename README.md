# 🪙 Customer Analytics & Churn Prediction Dashboard

An end-to-end data science project analyzing online retail data to extract business insights, segment customers, and predict churn using machine learning.

---

## 🚀 Live Demo
👉 https://retail-customer-intelligence-djcav4yvhsqkr9ez6r3tih.streamlit.app/

---

## 📊 Project Overview

This project transforms raw transactional data into actionable business insights.

It covers:
- Sales performance analysis
- Customer segmentation (RFM & Clustering)
- Churn prediction using machine learning
- Interactive dashboard using Streamlit

---

## 🎯 Business Objectives

- Understand sales trends and key revenue drivers  
- Identify high-value customers and at-risk customers  
- Predict customer churn to improve retention  
- Support decision-making with data-driven insights  

---

## 🧠 Key Features

### 📈 Sales Analytics
- Monthly & hourly revenue trends  
- Top products by quantity and revenue  
- Top countries by revenue  

### 👥 Customer Segmentation
- RFM Analysis (Recency, Frequency, Monetary)  
- Customer classification (VIP, Loyal, Regular, At Risk)  
- K-Means clustering  

### 🤖 Churn Prediction
- Machine Learning model (XGBoost)  
- Predict churn probability  
- Customer-level predictions  

### 📊 Interactive Dashboard
- Built using Streamlit  
- Dark premium UI  
- Real-time filtering  
- Customer lookup  

---

## 🛠️ Tech Stack
